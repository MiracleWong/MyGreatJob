- [用「三遍写作法」完成日更](https://mp.weixin.qq.com/s/-DKeoKaoi00s2OYw95Jn0g)
	> 「三遍写作法」是我完成「日更」的一个核心秘密。
	
	1. 灵感捕捉
		- 「好主题」最基本的条件是：这个话题是大家关心的且我正好言之有物、有感而发。
		- 灵感出现在的场景：共性是「有合适的环境」和「有新的信息输入」
			1. 与人聊天时
			2. 空闲独处时——睡觉前、起床后、刷牙时、上WC
			3. 看书或者听书时—— 所谓好书一般都需要看很多遍，每次都有不同的感受。
			4. 体验新产品时
			5. 他人提出好的问题时
		- 记录：手机备忘录、微信朋友圈、手机录音
	2. 流畅的扩充内容
		- 最好在灵感记录下来不久，就把这些灵感都搬到写作工具中去
		- 当开始写作时最好找此时此刻仍然有更多感慨的主题
		- 一定要确保自己在写作时不能中断，即一定要保证敲键盘输文字的速度。这个速度是思维流畅度的标志因素。
		- 「书写流畅度」体现出你对这个想法的「认同度」和你内心中与之相关素材的「丰富度」
	3. 最终写成文章
		- 「完成比完美重要」
		- 选「完成度」最高的那篇文章为基础，做逻辑梳理、文章结构优化和文字润色
